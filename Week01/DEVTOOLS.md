## 크롬 개발자 도구 시작하기

-   단축키 : F12, Cmd + Alt + I (Mac), Ctrl + Shift + I (Window)
-   크롬 우측 메뉴버튼 -> 도구 더보기 -> 개발자 도구
-   우클릭 -> 검사
